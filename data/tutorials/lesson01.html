Lesson 1: Getting Started

Welcome to ModelViewer!

ModelViewer is a powerful 3D model visualization application that supports various file formats including OBJ, STL, glTF, FBX, and CAD formats like STEP, IGES, and BREP.

This tutorial will guide you through all the features and help you become proficient with the application.

Screenshot: tutorial_01_main_window.png
The ModelViewer main window showing the 3D viewport, model tree, and control panels

Interface Overview

The ModelViewer interface consists of several key areas:

Step 1: Main Viewport
The large central area where your 3D models are displayed and manipulated. This is your primary workspace.
Step 2: Model Tree Panel (Right)
Shows every object in your scene organized as a hierarchical tree that mirrors the model's assembly structure — not a flat list. Selecting an assembly node selects all of its child parts, and checkboxes are tri-state: checking/unchecking an assembly propagates visibility to all its children, while a partially-visible assembly shows a dash. Right-click for context menu options including expand/collapse (1st level, all children, or all-collapse), rename (double-click, undoable), and visibility actions.
Step 3: Properties Panel (Right)
Below the model tree, contains tabs for Materials, Textures, Transformations, and Environment settings.
Step 4: View Toolbar (Bottom)
Auto-hiding toolbar at the bottom of the viewport with quick access to view manipulation, camera modes, and display settings.
Step 5: Menu Bar (Top)
Standard menu bar with File, Edit, Tools, Window, and Help menus for accessing all application features.
Most side panels — the Model Tree, Object Transformations, Predefined Materials, and Visualization Environment — can be detached into their own floating windows. Look for a detach icon in each panel's header.

The view toolbar automatically hides after 2 seconds. Move your mouse to the bottom edge of the viewport to reveal it again.

What You'll Learn

Throughout this tutorial, you will learn:

  • How to open and import 3D models
  • Navigate the 3D viewport using mouse and keyboard
  • Select and manipulate objects
  • Use different camera and display modes
  • Apply materials and textures
  • Configure lighting and environment
  • Work with visibility and organization
  • Use advanced features like clipping planes
  • Optimize performance for large models
